# カドゥケウスZ/NB RTAガイド

## はじめに

このページは「カドゥケウスZ」と「カドゥケウス NEW BLOOD」2作品のRTAのための情報を掲載しています。

しかし、カドゥケウスRTAにおいては攻略情報よりも**リモコン操作の習熟**が最も重要です。満足に操作できなければせっかくの情報を活かすことができません。

リモコンの操作については練習するしかありませんが、逆に言えば練習するほどタイムが目に見えて縮むゲームです。ここでは基本的なWiiリモコンの持ち方から解説しているので、まずは自分に合った操作環境を整えるところから初めましょう。
